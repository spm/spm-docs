# Contributing to this documentation

This documentation website is written in [Markdown](https://en.wikipedia.org/wiki/Markdown)
and compiled to HTML using a framework called [MkDocs](https://www.mkdocs.org/).

You can find the source code for this website in the [spm-docs Github repository](https://github.com/spm/spm-docs). If you wish to contribute text, please submit a pull request. Please ensure that you have first tested your contribution by building the documentation on your own computer - you can find instructions in [README.md](https://github.com/spm/spm-docs/blob/main/README.md).
